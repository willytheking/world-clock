# 🕐 Horloge Mondiale

Une application web interactive qui affiche l'heure en temps réel à New York et à Paris.

## 🌟 Fonctionnalités

- ⏰ Affichage en temps réel des heures
- 🗽 Heure à New York (fuseau horaire EST/EDT)
- 🗼 Heure à Paris (fuseau horaire CET/CEST)
- 📱 Design responsive et moderne
- 🎨 Interface élégante avec dégradés colorés

## 🚀 Utilisation

1. Clône le dépôt
2. Ouvre `index.html` dans ton navigateur
3. L'horloge se met à jour automatiquement chaque seconde

## 📁 Fichiers

- `index.html` - Structure HTML
- `style.css` - Styling et mise en page
- `script.js` - Logique JavaScript pour l'affichage de l'heure

## 🔧 Technologies

- HTML5
- CSS3
- JavaScript (Intl.DateTimeFormat API)
